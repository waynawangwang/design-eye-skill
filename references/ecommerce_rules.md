# 电商封面设计规范 (E-commerce Cover Design Spec)

本文档整合所有电商封面规则，供 SKILL.md 引用。

---

## 一、禁止项（违反即错误）

| 规则类别 | 禁止内容 | 说明 |
|---------|---------|------|
| **尺寸** | 禁止 1:1、800×800、正方形、产品主图比例 | 电商封面必须是横幅比例 |
| **背景** | 禁止纯白底、纯灰底、渐变底、影棚背景 | 必须是场景化环境 |
| **构图** | 禁止商品居中孤立、无环境、无文案区域 | 商品必须嵌入场景 |
| **视角** | 禁止微距产品摄影视角 | 必须是设计海报视角 |
| **文案** | 禁止"你为我补全""你定""你帮我想"等指令性文字 | 必须自动生成适配文案 |
| **格式** | 禁止"产品摄影""电商主图""白底图"等描述词 | 必须使用专业设计术语 |

---

## 二、必须项

| 规则类别 | 必须内容 | 说明 |
|---------|---------|------|
| **尺寸** | 横幅 16:9 或 3:2（如用户另行指定则以用户为准） | 默认横幅比例 |
| **背景** | 场景化环境（建筑空间/自然景观/抽象氛围场） | 禁止纯色或无环境背景 |
| **构图** | 商品嵌入场景，预留文案信息层级区域 | 确保文案可读性 |
| **视角** | 设计海报视角（平视/仰视/空间感） | 禁止产品摄影视角 |
| **文案** | 自动生成3-5字适配文案 | 文案需服务转化目的 |
| **格式** | 使用"海报设计""封面设计""平面设计" | 专业设计术语 |

## 三、信息层级 (L1-L4)

电商封面的信息必须按照以下层级组织：

```
L1: 主视觉（商品/场景）— 画面焦点，占比40-60%
L2: 价格信息 — 视觉冲击力，通常最大字号或特殊处理
L3: 主标题/副标题 — 情感连接，服务于"想买"的目的
L4: 品牌标识/信任标记 — 底部或角落，低调但存在
```

**层级应用规则**：
- 每个层级必须有明确的视觉区分（字号/色彩/位置）
- L1必须占据画面最大面积
- L2必须使用冲击色（5-10%占比的那个颜色）
- L3必须匹配主决策大师的风格签名
- L4必须低调但不消失

## 四、Prompt 强制词汇

### 4.1 禁止词汇（必须避免）

以下词汇**禁止**出现在最终 Prompt 中：
- ❌ 产品摄影
- ❌ 电商主图
- ❌ 白底图
- ❌ 微距
- ❌ 影棚
- ❌ 纯色背景
- ❌ 800×800
- ❌ 正方形
- ❌ 1:1

### 4.2 必须词汇（必须包含）

以下词汇**必须**出现在最终 Prompt 中：
- ✅ 海报设计
- ✅ 封面设计
- ✅ 平面设计
- ✅ 场景化
- ✅ [实际比例]（如 16:9 或 3:2）

### 4.3 排除规则（必须包含）

最终 Prompt 的排除规则中**必须包含**：
- 无白底
- 无孤立商品
- 无产品摄影视角
- 无影棚灯光

## 五、尺寸冲突解决规则

**优先级（从高到低）**：
1. **用户显式指定** > 领域默认规则
2. 当用户指定比例与领域默认冲突时：
   - 如果用户指定了"正方形"或"手机壁纸 9:16"，**尊重用户选择**
   - 但在输出中标注"注：用户指定尺寸 [X]，已偏离电商封面横幅推荐"
3. 当电商封面 + 用户未指定尺寸时：**输出横幅 16:9**（默认）
4. 非电商领域 + 用户未指定尺寸时：智能选择最适配比例
