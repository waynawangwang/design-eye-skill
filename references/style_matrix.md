# 风格判定矩阵 (Style Affinity Matrix)

| 编号 | 元风格 | 核心定义 | 视觉特征 |
|------|--------|---------|---------|
| M1 | 几何理性主义 | 以数学秩序构建视觉 | 网格、几何、结构、系统化 |
| M2 | 有机叙事 | 以自然流动承载叙事 | 曲线、手绘、文化符号、叙事密度 |
| M3 | 印刷与编辑 | 以排版作为空间架构 | 网格层级、字体结构、信息秩序 |
| M4 | 极简与虚空 | 以减法抵达本质 | 大留白、单一焦点、材料诚实 |
| M5 | 叙事与序列 | 以时间性构建空间 | 分镜、序列、时间线、格系统 |
| M6 | 混沌与张力 | 以破坏重组创造能量 | 拼贴、故障、冲突、解构 |

## 风格判定矩阵（速查，0-10分）

**分数定义**：矩阵中的分数为"综合亲和度评分 (HAS)"，为经验值而非三维度公式计算值。三维度加权（概念40% + 情绪35% + 领域25%）仅用于自由加权场景（混合属性主题）。

### 评分依据注释（新增）

- **国潮/东方**：概念上东方符号可被结构化(M3=6)或叙事化(M2=9)，情绪上文化自豪与手作温度压倒几何冷感(M1=4)，领域上海报/插画天然适配有机叙事
- **科技/未来**：概念上几何理性统治科技视觉(M1=9)，情绪上冷峻未来感排斥有机温暖(M2=2)，领域上UI/信息设计适配网格系统(M1=9)
- **自然/有机**：概念上自然形态排斥几何约束(M1=3)，情绪上温暖有机压倒冷峻理性(M4=6)，领域上插画/品牌视觉适配有机叙事(M2=9)

（其他行的评分依据注释可类似添加）

## 打分规则

- **预制分数定义**：矩阵中的分数为"综合亲和度评分 (HAS)"，为经验值而非三维度公式计算值
- **三维度加权**（概念亲和度40% + 情绪亲和度35% + 领域适配度25%）**仅用于自由加权场景**（混合属性主题）
- 最高分 = 主风格，第二高分 = 辅风格
- **风格融合策略**（替代"双主风格融合"）：

| 分差范围 | 融合模式 | 配比 | 影响范围 |
|-----------|----------|------|----------|
| < 1.0 | 强融合（双主风格） | 50:50 | 构图+色彩+隐喻 |
| 1.0-2.0 | 中融合（主+强辅） | 65:35 | 构图+色彩 |
| 2.0-3.5 | 弱融合（主+弱辅） | 80:20 | 仅影响色彩 |
| >= 3.5 | 单一主风格 | 100:0 | 辅风格仅作参考 |

## 主题属性动态判断指引（新增）

当主题属性不明确落在矩阵12行中时，使用动态判断：

1. **提取关键词**：从用户输入中提取核心概念
2. **分析三维度特征**：
   - 概念维度：抽象/具象、理性/感性、传统/现代
   - 情绪维度：安静/热烈、温暖/冷峻、喜庆/严肃
   - 领域维度：商业/艺术、平面/空间、静态/动态
3. **与元风格匹配**：将三维度特征与6大元风格匹配
4. **输出**：主题属性标签 + 三维度评分（0-10分）
5. **与矩阵对比**：将动态评分与矩阵预制分数（HAS）对比，取高分者

**优势**：
- 适应性更强：可处理任意主题
- 减少维护成本：不需要手动扩展矩阵
- 提升准确性：针对具体主题而非通用分类
