# 设计之眼 · 使用说明

> 丢一个主题进来，它帮你写「大师级」AI 生图提示词。中英双语，直接丢进豆包 / ChatGPT 出图。

---

## 快速开始

输入你想要的，越具体越好：

- 「设计一张咖啡馆开业海报」
- 「画一张赛博朋克城市概念图，尺寸 16:9」
- 「做一张微信文章封面，标题是《5条提示词超越90%的人》」

它会自动判断风格、选大师、给 Prompt。

---

## 6 大内置风格（可选）

想让风格精准一点，加个关键词就行：

| 你说 | 效果 |
|------|------|
| `M1` / 几何 / 理性 | 网格、秩序、专业感 |
| `M2` / 有机 / 叙事 | 手绘、故事感、东方韵味 |
| `M3` / 编辑 / 排版 | 杂志感、文字作主视觉 |
| `M4` / 极简 / 留白 | 干净、高级、呼吸感 |
| `M5` / 叙事序列 / 分镜 | 漫画、时间线、图像叙事 |
| `M6` / 混沌 / 冲击 | 拼贴、故障、视觉张力 |

例：「设计咖啡海报，走 M4 极简风」。

---

## 指定大师：在库的不在库的都能用

### 在库大师 → 精确还原

库内有 30+ 位各流派大师（由 4 项参数精确还原）。直接喊名字：

- 「用 田中一光 几何风」
- 「用 鸟山明 机械线条风」
- 「用 Barbara Kruger 红白黑宣言风」

### 不在库的大师 → AI 自动补全

你点的大师如果在库外（比如 **宫崎骏、安迪·沃霍尔、草间弥生**），skill 不会强行替换成别人，而是**由 AI 基于对该大师的了解自动补全风格描述**，照样出一份可用的 Prompt。输出中标明「库外大师（AI 知识补全）」以作区分。

### 自定义风格 → 同理

你写「赛博朋克」「蒸汽波」「像素风」「水墨动画」，这些不在 6 大元风格里也照样接——AI 会自行理解并生成匹配的视觉方案。

---

## 随机模式：想跳出套路

- 「随机模式，设计一张科技海报」→ 全库随机
- 「随机 M6，设计一张朋克封面」→ 只在 M6 池里随机

适合换口味、避免总撞同一两位大师。

---

## 尺寸控制

直接写比例，不写就按主题智能选：

- 「尺寸 2.35:1」（微信封面）
- 「竖版 3:2」「横版 16:9」

---

## 输出格式（每次四块）

| 块 | 内容 |
|------|------|
| 🎨 风格 | 用了什么元风格（或自定义风格） |
| 🎨 主角 | 画面焦点是什么、占多大 |
| 👨‍🎨 大师借鉴 | 参考了谁、各自管什么（库外大师会标注） |
| 📝 生图提示词 | 中文 + 英文，直接复制进生图工具 |

---

## 自动行为（不用管）

- 做电商 / 品牌 / 海报时，自动加一位「领域大师」管信息层级
- 你指定配色、或主题偏艺术 / 自然时，自动加一位「色彩大师」管配色
- 上述自动行为仅适用于在库的大师体系

---

## 几点提示

- **写真 / 人物类**：当「艺术海报」处理，不是摄影参数单；想要纯摄影精修得靠生图工具自身。
- **封面 / 海报类（带标题）**：内置大师偏向文字排版流，若想跳出可用「随机模式」或「走 M2 插画」。
- **完全跳出内置体系**：指定一个库外风格或大师，走 AI 知识补全通道。
- **遇到库内没有的大师**（如宫崎骏）：skill 不会拒绝也不会静默替换，而是 AI 补全后在输出中明确标注。
